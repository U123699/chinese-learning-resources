# 中文学习资源平台

一个简洁的HSK 1-6教材资源分享平台，支持响应式设计，适合手机和电脑使用。

## 功能特点

- 🎯 HSK 1-6级别分类
- 📱 响应式设计，支持手机和电脑
- 🔍 资源搜索和过滤
- 📥 直接下载资源
- 👨‍💼 管理员后台（模拟）
- 🚀 基于GitHub的文件托管

## 使用方法

### 1. 基本使用
1. 下载所有文件到本地
2. 用浏览器打开 `index.html`
3. 点击不同HSK级别查看资源
4. 使用搜索功能查找特定资源

### 2. 实际部署到GitHub
1. 创建一个GitHub仓库
2. 上传所有文件到仓库
3. 在仓库中创建资源文件夹结构：
4. 启用GitHub Pages：
- 进入仓库设置 → Pages
- 选择分支（如main）和根目录
- 保存后访问提供的URL

### 3. 添加真实资源
1. 将PDF、MP3等文件上传到GitHub仓库
2. 获取文件的"raw"链接
3. 在 `script.js` 的 `resourcesData` 数组中添加资源信息
4. 或创建 `resources.json` 文件并修改代码从该文件加载

## 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 交互逻辑
- `admin.html` - 管理员后台
- `README.md` - 说明文档

## 自定义修改

1. 修改颜色主题：编辑 `style.css` 中的颜色值
2. 添加更多资源：更新 `script.js` 中的 `resourcesData` 数组
3. 更改布局：调整 `style.css` 中的网格和间距
4. 添加语言支持：复制文本并翻译

## 注意事项

- 确保资源文件有合适的版权许可
- 大文件建议使用Git LFS
- 实际部署时建议添加访问统计
- 可考虑使用GitHub API自动更新资源列表
